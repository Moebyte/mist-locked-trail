# 雾锁迷踪 · Mist-Locked Trail

**民国三十七年 · 暮秋 · 上海 — 一桩失踪案 · 多条岔路 · 十一个结局**

一场发生在 1948 年深秋上海法租界的文字侦探冒险。私家侦探沈先生接下一桩女学生失踪案，从圣约翰大学、巡捕房、苏家与薛华立路 22 号之间抽丝剥茧，逐步牵出光华小学坠楼案、福生仓暗室与法租界背后的灰色利益网。

🔗 **在线游玩：** https://moebyte.github.io/mist-locked-trail/

## 玩法特色

- **纯文字侦探冒险**：以分支选择、线索收集和场景调查推进故事。
- **线索与道具举证**：在关键人物面前出示照片、信件、发夹、当票、货运单等物件，改变对话和后续走向。
- **三层推理谱**：围绕陈明远坠楼、陆小姐与赵先生、福生仓真相逐层推进。
- **路线时机与轻后果**：过早进入某些地点不会直接失败，但可能错过会面、只拿到弱证词，或让案卷走向不完整收束。
- **人物信任机制**：苏家线、银发夹、医院证词等会影响后期救援与结局完整度。
- **多结局结构**：包含拒接、归档、错误指认、情感误判、单救、双救、隐藏与真隐藏等 11 个结局。
- **关系图与推理墙**：辅助整理人物、线索与证据链。
- **自动存档与移动端适配**：适合桌面和手机浏览器游玩。

## 叙事设计

《雾锁迷踪》不是“选错即死”的路线游戏，而是让玩家在不同时间进入不同地点，得到不同版本的事实。

有些线索不会消失，但会变得不够有力；有些人物不会拒绝你，却会暂时保留；有些会面不是永远在那里等你。玩家最终写进案卷的，可能是真相，也可能只是一个更容易被接受的说法。

当前主要路线包括：

```text
茶馆接案
→ 圣约翰大学 / 苏家 / 巡捕房
→ 薛华立路 22 号与 203 室
→ 光华小学
→ 第二段推理
→ 福生仓行动
→ 医院与证词选择
→ 傅启元后巷交易
→ 第三段推理与结案
```

## 项目结构

```text
index.html                 # 页面结构
src/styles.css             # 界面样式与氛围效果
src/engine.js              # 游戏引擎、存档、举证、推理、关系图、时间压力
src/story.js               # 基础剧情节点、分支、结局
src/story-modules.js       # 剧情模块加载入口
src/story-modules/         # 路线修正、结局收束、证据面板、诊断工具等模块
scripts/                   # smoke 测试与可达性检查脚本
```

## 本地运行

```bash
# 浏览器直接打开
open index.html

# 或启动简易服务器
python3 -m http.server 8080
```

然后访问：

```text
http://localhost:8080/
```

## 开发与诊断

开发模式可通过 URL 参数开启：

```text
?dev=1
```

开发面板支持导出当前剧情诊断包，用于检查：

- 当前场景与可选项
- 已获得线索、道具、联系人和 flags
- 推理题注册与缺失条件
- 福生仓、医院、真相完整度等后期指标
- 当前路线尾迹与结局可达性

项目内包含若干 smoke 脚本，用于检查关键结局、苏家信任门、福生仓入口、推理注册表等是否仍然可达。
