# UI 美化与日间 / 夜间模式

## 背景

当前界面功能已经逐步完善，但视觉上仍偏工程化：

- 卡片层次不够；
- 按钮像调试按钮；
- 线索簿像管理面板，不像侦探档案册；
- 只有暗色风格，缺少日间阅读模式。

本次改造目标是让界面更接近一款文字侦探游戏，而不是纯工具页面。

---

## 本次改动

### 1. 增加日间 / 夜间模式

复用顶部原来的“氛围”按钮，改为主题切换：

```text
☀️ 日间
🌙 夜间
```

主题选择会保存到：

```text
localStorage: mlt_theme_mode
```

刷新页面后保留用户上次选择。

---

## 夜间模式

夜间模式定位为：

```text
侦探事务所 / 雨夜 / 暗色玻璃卡片
```

特点：

- 深蓝黑背景；
- 金色标题与关键按钮；
- 卡片使用轻微玻璃质感；
- 保留雨夜与雾气氛围。

---

## 日间模式

日间模式定位为：

```text
旧报纸 / 档案馆 / 纸张阅读
```

特点：

- 米白纸张背景；
- 棕金色标题；
- 卡片像档案纸片；
- 更适合长时间阅读。

---

## 视觉优化范围

本次统一优化：

- 顶部按钮；
- 当前场景卡片；
- 翻页按钮；
- 选择按钮；
- 右侧线索簿；
- 总览卡片；
- 线索 / 物品 / 人物卡片；
- 手机底部操作栏；
- Toast 提示。

---

## 不改动的内容

本次只改 UI 和主题，不改：

- 剧情节点；
- 存档结构；
- 推理逻辑；
- 举证逻辑；
- 结局分流。

---

## 后续建议

如果继续提升视觉品质，下一步可以做：

- 首页封面视觉；
- 章节过场页；
- 线索获得动画；
- 结局页专属排版；
- 人物关系图重新设计。
