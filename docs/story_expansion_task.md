# 雾锁迷踪 · 文本扩写任务

## 目标
将全游戏文本从目前的 ~1.6万字扩写到 ~3万字，重点增加场景描写、角色对话、人物性格、气氛铺陈。

## 扩写规则

1. **只写文本内容**（纯文字），不写JS代码，不修改节点结构
2. **必须保留所有 <span class="sys">"..."</span> 格式（角色说话风格）
3. **必须保留所有 <b>...</b> 格式（角色名强调）
4. **必须保留所有现有线索/道具/结局的触发条件不变**
5. **新增内容不能破坏已有的 goto 链条**
6. **按节点ID组织，每个节点一个章节**

## 格式要求

每个节点写为：

### [节点ID] - [当前标题]
> 当前字数: XXX → 目标字数: XXX

[扩写后的完整文本，包含原文保留+新增内容]
