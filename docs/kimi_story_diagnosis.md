# 《雾锁迷踪》剧本诊断报告

> 诊断人：编辑部总编 Kimi | 2026-06-16

## 一、人物塑造薄弱

### 1. 苏晚亭：缺「声音」、缺「选择」、缺「复杂性」
苏晚亭是全剧的麦高芬（失踪者），但从来没有真正「活」过。她的性格标签只有：懂事、倔强、功课好。剧本通过四五个节点反复拼凑她——舍监说她哭过、母亲说她淋雨回家、陈明远叫她「晚亭吾爱」——但这些都是**他人视角的证词**，不是她自己的行动。

**致命缺陷**：她为什么哭？剧本从未直接回答。是恐惧？愧疚？爱情？发现真相后的崩溃？这个缺失让她只剩下功能性——为引路而存在。

**P0 修改建议**：`ch2_univ_paper` 的铅笔清单可以加入一段内心独白草稿，而不是只有地址和时间。

### 2. 陈明远：功能性死者，内心矛盾为零
陈明远是「发现真相→被灭口」的经典工具人。遗书「愧对学生，无颜苟活」是**陈词滥调**。

**问题**：他发现了走私为什么不报警？当了三百大洋用去哪了？和苏晚亭是爱情还是利用？

**P0 修改建议**：`ch3_chen_letter` 是全剧最关键的文本，但写得像剧情摘要而不是情书。需要具体细节让他「告诉」玩家他死前看到的那一夜。

### 3. 陆念薇：动机前后打架
- `end_boss_lu`：主谋，心狠手辣
- `end_conspiracy_detail`：中间人，上面还有人
- `ch2_203_search`：她也收到恐吓信，也是受害者

**建议**：统一为核心动机——「既是加害者也是受害者」，被公董局胁迫，同时用 300 大洋试图赎身。

### 4. 周明远 & 沈玉兰
- 周明远交完定金后消失。建议他有秘密，甚至可做第二嫌疑人
- 沈玉兰茶楼惊艳出场后蒸发。建议第三幕有她的行动线

### 5. 沈玉芳
- 被关近一个月但心理状态过于冷静
- 和陈明远「走得近」没有进一步展开

## 二、场景结构雷同

**核心问题：8 个节点共享「走进去→听人说话→拿线索→去下一处」模板**

| 节点 | 结构 |
|------|------|
| ch2_univ_matron | 问舍监→大段回忆→得线索 |
| ch2_univ_door | 问门房→大段回忆→得线索 |
| ch2_home_talk | 问母亲→大段回忆→得线索 |
| ch2_ask_landlord | 问老头→大段回忆→得线索 |
| ch3_school 系列 | 连续四个问吴校长 |

**P0 建议**：光华小学四个节点合并为一次「校长办公室对峙」，根据玩家态度给出不同深度的信息。

## 三、第三幕问题
1. **转折生硬**：缺少紧迫感
2. **潜入无张力**：没有中间状态
3. **幕后线太抽象**：公董局的人从未出现
4. **反派缺席**：第三幕一个人搜仓库
5. **隐藏结局太理想化**：民国法租界不可能这么顺利
